# Plan 17-03: 에디터 애니메이션/글꼴/색상 추가 (TEMPLATE-EDIT-01, 02, 03)

## Tasks

### Task 1: Section별 애니메이션 설정 (TEMPLATE-EDIT-01)

**Files to Modify:**
- `src/types/template.ts` — Section 인터페이스에 animation 필드 추가
- `src/components/templates/editor/StyleEditor.tsx` — Section 스타일 설정 UI
- `src/components/templates/engine/TemplateEngine.tsx` — animation class 적용
- `src/app/globals.css` — animation classes 추가

**Implementation:**
1. Section 인터페이스에 style.animation 필드 추가:
   ```typescript
   interface Section {
     style?: {
       animation?: 'fade-in' | 'slide-up' | 'slide-left' | 'slide-right' | 'bounce' | 'scale-up' | 'none';
       animationDuration?: number;
       animationDelay?: number;
     };
   }
   ```
2. StyleEditor에 애니메이션 선택 UI 추가 (Select 컴포넌트)
3. TemplateEngine에서 Section.style.animation → CSS class 매핑:
   - `fade-in` → `.animate-fade-in`
   - `slide-up` → `.animate-slide-up`
   - `bounce` → `.animate-bounce`
   - 등등
4. CSS classes 추가 (globals.css):
   ```css
   @keyframes fade-in { from { opacity: 0; } to { opacity: 1; } }
   @keyframes slide-up { from { opacity: 0; transform: translateY(20px); } to { opacity: 1; transform: translateY(0); } }
   @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-10px); } }
   ```
5. 실시간 preview — 선택 즉시 preview에 적용

### Task 2: Section별 글꼴 설정 (TEMPLATE-EDIT-02)

- `src/components/templates/editor/StyleEditor.tsx` — 폰트 선택 UI
- `src/components/templates/engine/TemplateEngine.tsx` — fontFamily 적용

**Implementation:**

3. StyleEditor에 폰트 선택 UI 추가
4. TemplateEngine에서 Section.style.fontFamily → CSS `font-family` 적용
5. Google Fonts CDN 동적 로드 (Phase 11 fonts.ts 활용)

### Task 3: Section별 색상 테마 설정 (TEMPLATE-EDIT-03)

**Files to Modify:**
- `src/types/template.ts` — Section 스타일에 색상 필드 추가
- `src/components/templates/editor/StyleEditor.tsx` — 색상 선택 UI
- `src/components/templates/engine/TemplateEngine.tsx` — 색상 적용

**Implementation:**
