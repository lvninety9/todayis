# CSS Scroll-Driven Animation Reference
# - animation-timeline: scroll() — scroll position-linked effects
# - animation-timeline: view() — scroll-triggered reveals (no IntersectionObserver)
# - Chrome 115+, Safari 18+, Firefox 133+ 지원
# - Progressive enhancement: 미지원 브라우저에서 graceful degradation
# - 성능: compositor thread에서 실행, INP 영향 없음
</context>



    2. 각 카드에 staggered animationDelay: index * 50ms (0, 50, 100, 150...)

    Before:
    ```tsx
    <div className="grid grid-cols-1 ...">
      {templates.map((template) => (
        <TemplateCard key={template.id} ... />
      ))}
    </div>
    ```

    After:
    ```tsx
    <div 
      className="grid grid-cols-1 ..."
      style={{
        animation: 'fade-in linear both',
        animationTimeline: 'view()',
        animationRange: 'entry 0% cover 50%'
      }}
    >
      {templates.map((template, index) => (
        <div 
          key={template.id} 
          style={{ animationDelay: `${index * 50}ms` }}
          className="animate-fade-in-up"
        >
          <TemplateCard ... />
        </div>
      ))}
    </div>

    {/* Add to globals.css or local <style> */}
    <style jsx>{`
      .animate-fade-in-up {
        opacity: 0;
        animation: fade-in-up 0.5s ease-out both;
      }
      @keyframes fade-in-up {
        from { opacity: 0; transform: translateY(20px); }
        to { opacity: 1; transform: translateY(0); }
      }
    `}</style>
    ```

    Tailwind의 fade-in, slide-up 클래스 활용 가능 (tailwindcss-animate 설치됨).
