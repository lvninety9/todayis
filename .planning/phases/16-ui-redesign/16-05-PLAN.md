settings/page.tsx, admin/page.tsx 테이블을 다음과 같이 업데이트:

```tsx
import { Table, TableBody, TableCell, TableHead, TableHeader, TableRow } from '@/components/ui/table';

<Table>
  <TableHeader>
    <TableRow>
      <TableHead>이름</TableHead>
      <TableHead>이메일</TableHead>
      <TableHead>역할</TableHead>
      <TableHead>가입일</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>

      <TableRow key={user.id}>
        <TableCell>{user.name}</TableCell>
        <TableCell>{user.email}</TableCell>
        <TableCell>{user.role}</TableCell>
        <TableCell>{user.createdAt}</TableCell>
      </TableRow>
    ))}
  </TableBody>
</Table>
```

2. 페이지네이션:
```tsx
<TablePagination
  total={totalUsers}
  page={page}
  pageSize={pageSize}
  onPageChange={setPage}
/>
```

3. 검색:
```tsx
<Input
  placeholder="검색..."
  value={search}
  onChange={e => setSearch(e.target.value)}
/>
```
</action>
</task>

## Task 2: Micro-interactions (UI-11)

<task>
<read_first>
- src/app/globals.css
- src/components/ui/button.tsx
</read_first>

<acceptance_criteria>
- hover/focus/transition이 모든 컴포넌트에 일관되게 적용
- 로딩 상태가 모든 액션에 명확히 표시
- 성공/에러 피드백이 시각적으로 명확히
</acceptance_criteria>

<action>
globals.css 또는 button.tsx에 Micro-interactions 추가:

1. Hover/Active transitions:
```css
.button {
  transition: all var(--duration-200) ease;
}

.button:hover {
  transform: translateY(-1px);
  box-shadow: var(--shadow-md);
}

.button:active {
  transform: translateY(0);
}
```

2. Focus ring:
```css
.button:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}
```

3. Loading state:
```tsx
{isLoading && <Spinner className="animate-spin" />}
```

4. Success/Error feedback:
```tsx
{success && <p className="text-green-500">저장되었습니다</p>}
{error && <p className="text-red-500">오류가 발생했습니다</p>}
```
</action>
</task>

## Task 3: Accessibility (UI-12)

<task>
<read_first>
- src/components/ui/button.tsx
- src/components/ui/input.tsx
- src/components/ui/card.tsx
</read_first>

<acceptance_criteria>
- 모든 버튼에 aria-label 또는 aria-labelledby
- 모든 입력 필드에 associated labels
- focus indicators가 명확히 보임
- skip links 추가 (선택적)
</acceptance_criteria>

<action>
components/ui에 Accessibility 추가:

1. Button aria-label:
```tsx
<Button aria-label="템플릿 편집">
  <EditIcon />
</Button>

<IconButton aria-label="메뉴 열기" onClick={toggleMenu}>
  <MenuIcon />
</IconButton>
```

2. Input association:
```tsx
<div className="space-y-2">
  <Label htmlFor="email">이메일</Label>
  <Input id="email" type="email" aria-describedby="email-error" />
  <p id="email-error" className="text-sm text-red-500">
    {errors.email}
  </p>
</div>
```

3. Focus indicators:
```css
*:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}
```

4. ARIA live regions:
```tsx
<div aria-live="polite" aria-atomic="true">
  {message && <p>{message}</p>}
</div>
```

5. Skip links (layout.tsx):
```tsx
<a href="#main" className="sr-only focus:not:sr-only">
  메인으로 건너뛰기
</a>
<main id="main">
  {/* Page content */}
</main>
```
</action>
</task>
